# :soccer: CLI Campeonato Catarinense

Ferramenta de linha de comando que retorna informações do Campeonato Catarinense

# Instalação

`npm install -g cli-catarinense`

# Uso

## Classificação

`catarinense classificacao` ou `catarinense c`
